# Users & Accounts PRD

## Goals

Enable free account creation and secure authentication.

## Functional Requirements

-   User registration
-   Login / Logout
-   Password hashing
-   Basic profile (display_name)
-   Lobby creation
-   Lobby invitation via token
-   Role enforcement (DM vs Player)

## Acceptance Criteria

-   User can register and log in
-   DM can create lobby
-   Invited user can join via token
-   Role-based access works correctly
