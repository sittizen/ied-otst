# Calendar Product Requirements Document

## Goals

Allow DMs to schedule game sessions and players to RSVP.

## Functional Requirements

-   Create session (DM)
-   Edit / cancel session
-   RSVP (yes \| no \| maybe)
-   List upcoming and past sessions

## Acceptance Criteria

-   DM creates session
-   Players RSVP
-   Status visible to all members
